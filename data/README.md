
## Details

- **data/**  
  Enthält die SQLite-Datenbankdatei. Wird beim ersten Start automatisch angelegt.  

- **uploads/**  
  Hier werden alle hochgeladenen Bilder der 3D-Drucke gespeichert. Auch dieser Ordner wird automatisch erstellt.  

> Hinweis: Beide Ordner sind **leer bei Projektstart** und müssen nicht manuell angelegt werden, außer du willst sie versionieren.
